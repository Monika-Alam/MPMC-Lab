# MPMC Lab Programs (8086 Assembly Language)

Overview:
This repository contains basic 8086 Assembly Language programs developed as part of the Microprocessor and Microcontroller Programming (MPMC) laboratory course.

The programs demonstrate fundamental assembly language concepts including:

- Keyboard Input and Output
- Arithmetic Operations
- Conditional Statements
- Looping
- Pattern Generation
- Bitwise Shift Operations
- Rotate Operations

All programs use DOS Interrupt 21H services for input/output operations.

Programs Included

Problem 1: Character Echo
File: `prb1.asm`

Reads a character from the keyboard and immediately displays the same character on the screen.

Concepts:
- Keyboard Input
- Character Output

Problem 2: Addition of Two Numbers
File: `prb2.asm`

Takes two single-digit numbers as input and displays their sum.

Concepts:
- Arithmetic Addition
- ASCII to Numeric Conversion


Problem 3: Subtraction of Two Numbers
File: `prb3.asm`

Accepts two single-digit numbers and displays the subtraction result.

Concepts:
- Arithmetic Subtraction
- Data Conversion

Problem 4: Multiplication of Two Numbers
File: `prb4.asm`

Multiplies two single-digit numbers and displays the result.

Concepts:
- MUL Instruction
- Arithmetic Operations


Problem 5: Division of Two Numbers
File: `prb5.asm`

Performs division between two single-digit numbers and displays the quotient.

Concepts:
- DIV Instruction
- Quotient Calculation

Problem 6: Sum of Three Numbers
File: `prb6.asm`

Takes three numbers as input and calculates their total sum.

Concepts:
- Loop Instruction
- Accumulation


Problem 7: Smallest of Two Numbers
File: `prb7.asm`

Compares two numbers and displays the smaller value.

Concepts:
- CMP Instruction
- Conditional Jump

Problem 8: Smallest of Three Numbers
File: `prb8.asm`

Finds and displays the smallest number among three inputs.

Concepts:
- Multiple Comparisons
- Decision Making

Problem 9: Largest of Three Numbers
File: `prb9.asm`

Finds and displays the largest number among three inputs.

Concepts:
- Comparison Operations
- Conditional Branching

Problem 10: Star Pattern Generation
File: `prb10.asm`

Prints a decreasing star (`*`) pattern using nested loops.

Example:

****
***
**

Concepts:
- Nested Loops
- Pattern Printing

Problem 11: Shift Left Operation
File: `prb11.asm`

Performs a logical left shift (`SHL`) on an input number and displays the result.

Concepts:
- Bitwise Shift Left
- Binary Operations


Problem 12: Shift Right Operation
File: `prb12.asm`

Performs a logical right shift (`SHR`) on an input number and displays the result.

Concepts:
- Bitwise Shift Right
- Binary Operations


Problem 13: Rotate Left Operation
File: `prb13.asm`

Rotates the bits of a number to the left using the `ROL` instruction.

Concepts:
- Rotate Instructions
- Bit Manipulation

Problem 14: Rotate Right Operation
File:`prb14.asm`

Rotates the bits of a number to the right using the `ROR` instruction.

Concepts:
- Rotate Instructions
- Bit Manipulation


Tools and Environment

- 8086 Assembly Language
- DOS Interrupt 21H
- EMU8086 


After completing these programs, students will understand:

- Basic Assembly Language Syntax
- Register Usage
- DOS Interrupt Services
- Arithmetic Instructions
- Comparison and Branching
- Loop Structures
- Bitwise Operations
- Pattern Generation Techniques
